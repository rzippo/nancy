Curve

Class of functions on which NetCal operators are defined. They are piecewise affine and ultimately pseudo-periodic. Pseudo-periodic means that f(t+d)=f(t)+cf(t + d) = f(t) + cf(t+d)=f(t)+c, where ccc is the step gained after each pseudo-period. Ultimately means that the function has such property for t≥Tt \ge Tt≥T.

note

Implementation of data structure described in [BT08] Section 4.1
