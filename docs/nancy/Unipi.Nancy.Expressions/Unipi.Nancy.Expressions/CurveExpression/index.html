CurveExpression

Class which describes NetCal expressions that evaluate to curves. The class aims at providing the main methods to build, manipulate and print network calculus expressions.

public abstract class CurveExpression : Unipi.Nancy.Expressions.Internals.IGenericExpression<CurveExpression>
