RationalExpression

The class aims at providing the main methods to build, manipulate and print expressions which evaluate to rational numbers and are based on NetCal curves.

ApplyEquivalence(Equivalence, CheckType)​

Applies an equivalence to the current expression.

public RationalExpression ApplyEquivalence(Equivalence equivalence, CheckType checkType)

Parameters​

equivalence Equivalence
The equivalence to be applied to (a sub-part of) the expression.

checkType CheckType
Since the equivalence is described by a left-side expression and a right-side expression, this parameter identifies the direction of application of the equivalence (match of the left side, and substitution with the right side, or vice versa, or both).

Returns​

RationalExpression
The new equivalent expression if the equivalence can be applied, the original expression otherwise.
