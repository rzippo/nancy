Plot(IReadOnlyCollection<Curve>, String, Nullable<Rational>)​

Plots a set of curves. It attempts to parse the names for the curves from the optional parameter names or from the expression used for the curves parameter. Otherwise, the curves will be given default names f, g, h and so on.

note

The names of the curves can be automatically captured if one uses a syntax like "Plots.Plot([b1, b2, b3], upTo: 10);" Note that this works if each curve has its own variable name, rather than being the result of an expression.

public static void Plot(IReadOnlyCollection<Curve> curves, string names, Nullable<Rational> upTo)

Parameters​

curves IReadOnlyCollection<Curve>

names String

upTo Nullable<Rational>

The x-axis right edge of the plot. If null, it is set by default as maxi(Ti+2∗di)max_{i}(T_i + 2 * d_i)maxi​(Ti​+2∗di​).
