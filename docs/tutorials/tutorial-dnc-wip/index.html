Tutorial for Deterministic Network Calculus

This tutorial shows basic concepts and results from DNC, with a focus on how to use them in practice.

We show this in two ways:

  • scripts using the MPPG syntax, supported by RTaW min-plus playground and nancy-playground
  • C# file-based apps using Nancy

Single node, single flowâ€‹

We start from the simplest case, a single node traversed by a single flow of traffic.

For the flow of traffic, we use a (maximal) arrival curve: it is a function that upper bounds the amount of traffic that can arrive in any interval of time (see [Dnc18, Section 5.1]). A common type is the token-bucket arrival curve.

For the node, we use a (minimal) service curve: it is a function that lower bounds the amount of backlogged traffic that the node will serve in any interval of time (see [Dnc18, Section 5.2]). A common type is the rate-latency service curve.

Combining these two characterizations, we can compute worst-case delay and backlog bounds (see [Dnc18, Section 5.3]).

  • MPPG
  • C#
// arrival curve for the traffic
ac := bucket(2, 2)
// service curve for the traversed node
sc := ratency(3, 1)
// plot the two curves
plot(ac, sc)
// worst-case delay bound computed with horizontal deviation
hdev(ac, sc)
// worst-case backlog bound computed with vertical deviation
vdev(ac, sc)
#:package Unipi.Nancy@1.3.0
#:package Unipi.Nancy.Plots.ScottPlot@1.0.4

using System.Globalization;
using System.IO;
using Unipi.Nancy.NetworkCalculus;
using Unipi.Nancy.MinPlusAlgebra;
using Unipi.Nancy.Numerics;
using Unipi.Nancy.Plots.ScottPlot;

CultureInfo.CurrentCulture = CultureInfo.InvariantCulture;

// arrival curve for the traffic
var ac = new SigmaRhoArrivalCurve(2, 2);
// service curve for the traversed node
var sc = new RateLatencyServiceCurve(3, 1);
// plot the two curves
var plotBytes = ScottPlots.ToScottPlotImage([ac, sc]);
var plotTmpPath = Path.GetTempPath() + Guid.NewGuid().ToString() + ".png";
Console.WriteLine(plotTmpPath);
File.WriteAllBytes(plotTmpPath, plotBytes);
// worst-case delay bound computed with horizontal deviation
Console.WriteLine(Curve.HorizontalDeviation(ac, sc));
// worst-case backlog bound computed with vertical deviation
Console.WriteLine(Curve.VerticalDeviation(ac, sc));

Two nodes in tandemâ€‹

We now consider a single traffic flow traversing two nodes, one after the other. To compute a bound for the worst-case delay, a first approach is to compute per-node bounds and then sum them. However, to compute the bound on the second node we first need the output arrival curve after the first node (see [Dnc18, Section 5.3.2]). However, an important result of DNC is that we can do better than this, by exploiting Pay-Burst-Only-Once: we can concatenate the two service curves, using (min,+) convolution, and obtain a lower WCD bound.

  • MPPG
  • C#
// arrival curve for the traffic
ac := bucket(2, 2)
// service curves for the two nodes traversed
sc_1 := ratency(3, 1)
sc_2 := ratency( 3, 1)
// First idea: compute WCD per node, then sum them
// To do so, we first compute the *output arrival curve* between node 1 and 2
out_ac_1 := (ac / sc_1) /\ delay(0)
wcd_1 := hdev(ac, sc1)
wcd_2 := hdev(out_ac_1, sc2)
wcd_sum := wcd_1 + wcd_2
wcd_sum
// Better idea: use Pay-Burst-Only-Once
// Compute a service curve for the concatenation of the two nodes using (min,+) convolution
wcd_pboo := hdev(ac, sc1 * sc2)
wcd_pboo
#:package Unipi.Nancy@1.3.0
#:package Unipi.Nancy.Plots.ScottPlot@1.0.4

using System.Globalization;
using System.IO;
using Unipi.Nancy.NetworkCalculus;
using Unipi.Nancy.MinPlusAlgebra;
using Unipi.Nancy.Numerics;
using Unipi.Nancy.Plots.ScottPlot;

CultureInfo.CurrentCulture = CultureInfo.InvariantCulture;

// arrival curve for the traffic
var ac = new SigmaRhoArrivalCurve(2, 2);
// service curves for the two nodes traversed
var sc_1 = new RateLatencyServiceCurve(3, 1);
var sc_2 = new RateLatencyServiceCurve(3, 1);
// First idea: compute WCD per node, then sum them
// To do so, we first compute the *output arrival curve* between node 1 and 2
var out_ac_1 = Curve.Minimum(( Curve.Deconvolution(ac, sc_1) ), new DelayServiceCurve(0));
var wcd_1 = Curve.HorizontalDeviation(ac, sc_1);
var wcd_2 = Curve.HorizontalDeviation(out_ac_1, sc_2);
var wcd_sum = wcd_1 + wcd_2;
Console.WriteLine(wcd_sum);
// Better idea: use Pay-Burst-Only-Once
// Compute a service curve for the concatenation of the two nodes using (min,+) convolution 
var wcd_pboo = Curve.HorizontalDeviation(ac, Curve.Convolution(sc_1, sc_2));
Console.WriteLine(wcd_pboo);

Contended node, residual service curveâ€‹

We show now how to address a single node contended between multiple flows. We focus on the blind multiplexing case, i.e., where we do know the specific multiplexing policy that the server applies. We only assume to know a strict service curve for the aggregate, (see [Dnc18, Section 7.2.1]).

  • MPPG
  • C#
// arrival curves for two traffic streams on the same node 
a1 := bucket(2, 2)
a2 := bucket(2, 2)
// *strict* service curve offered by the node to the aggregate traffic
sc := ratency(5, 1)
// residual service curve: what is offered by the node to a1, assuming a2 has first taken as much service as it can
rsc_1 := nnupclosure(sc - a2)
plot(a1, a2, sc, rsc_1)
hdev(a1, rsc_1)
#:package Unipi.Nancy@1.3.0
#:package Unipi.Nancy.Plots.ScottPlot@1.0.4

using System.Globalization;
using System.IO;
using Unipi.Nancy.NetworkCalculus;
using Unipi.Nancy.MinPlusAlgebra;
using Unipi.Nancy.Numerics;
using Unipi.Nancy.Plots.ScottPlot;

CultureInfo.CurrentCulture = CultureInfo.InvariantCulture;

// arrival curves for two traffic streams on the same node 
var a1 = new SigmaRhoArrivalCurve(new Rational(2), new Rational(2));
var a2 = new SigmaRhoArrivalCurve(new Rational(2), new Rational(2));
// *strict* service curve offered by the node to the aggregate traffic
var sc = new RateLatencyServiceCurve(new Rational(5), new Rational(1));
// residual service curve: what is offered by the node to a1, assuming a2 has first taken as much service as it can
var rsc_1 = (sc - a2).ToNonNegative().ToUpperNonDecreasing();
Console.WriteLine(Curve.HorizontalDeviation(a1, rsc_1));
