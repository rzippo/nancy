Nancy's main types are:

  • Point, a single time-value pair, and Segment, defined in open interval ]a,b[]a, b[]a,b[
  • Sequence, which is piecewise affine function defined in a limited interval (can be opened, closed, a mix of both), constructed as a series of Points and Segments.
  • Curve, which is the representation of a UPP function, constructed as a Sequence defined in [0,T+d[[0, T + d[[0,T+d[ and the three parameters PseudoPeriodStart (T)(T)(T), PseudoPeriodLength (d)(d)(d) and PseudoPeriodHeight (c)(c)(c).
